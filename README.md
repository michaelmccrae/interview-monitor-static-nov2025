## Interview Monitor

A real-time tool for monitoring live interviews. It analyzes the conversation as it unfolds and provides automated insights.

### Features
- **Error Checking** – Flags incorrect facts in real time.
- **Suggested Follow-up Questions** – Generates context-aware prompts for the interviewer.
- **Automatic Term Lookup** – Identifies key terms and fetches relevant reference information.
- **Participant Metrics** – Tracks speaking time, topic distribution, and engagement patterns.

### Tech Stack
- **Framework:** Next.js  
- **UI:** React  
- **Deployment:** Vercel (recommended)
